## About

I build tools and write guides focused on **reducing digital linkability** — device hardening, network isolation, and privacy-preserving software.

No tinfoil hats. No paranoia. Just practical, layered threat reduction.

## Projects

### 🔌 Revenge Plugins
> Privacy-focused plugins for the Revenge/Vendetta Discord mod

| Plugin | What it does |
|--------|-------------|
| **MoreAlts** | Secure multi-account switcher — AES-256 encrypted token storage, PIN lock, biometrics |
| **GhostMode** | Go invisible with one tap in the You Bar — blocks typing indicators & read receipts |
| **TokenGuard** | Monitors every network request — alerts when a plugin sends your token to an unknown server |
| **Proxifier** | Routes Discord through a custom proxy — hides your real IP, bypasses network blocks |

→ [cordinsanity/revenge-plugins](https://github.com/cordinsanity/revenge-plugins)
